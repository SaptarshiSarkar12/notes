# Sankey

Sankey is used to depict flow from one data to another. Here's a simple example:

```mermaid
sankey-beta
  day,sleep,8
  day,work,10
  day,fun,2
  day,rest,4
```

```mermaid
sankey-beta
  day,sleep,8
  day,work,10
  day,fun,2
  day,rest,4
```
